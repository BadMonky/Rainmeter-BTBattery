# Release Checklist

- Build `Release|x64`.
- Build `Release|Win32`.
- Copy x64 DLL to `Release\x64\BTBattery.dll`.
- Copy x86 DLL to `Release\x86\BTBattery.dll`.
- Test x64 DLL in Rainmeter.
- Confirm `Examples\BTBatteryExample.ini` still loads.
- Confirm `LICENSE.md` is included in the release.
- Tag the release in GitHub.
